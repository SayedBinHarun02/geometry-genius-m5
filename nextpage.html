Asked Questions and Their Answers

Document Object Model is referred to by the abbreviation DOM. It is the internal representation of a web page by a web browser.The DOM decides what content belongs on a page and how each content piece interacts with other components. when we use JavaScript to add, remove, and modify elements of a website is called DOM manipulation. It is very common in web development.

JavaScript's DOM Selectors are used to choose specific HTML components from a page. There are five different methods to use selectors to choose items in a DOM. They are , 1.getElementsByTagName() 2.getElementsByClassName() 3.getElementById() 4.querySelector() 5.querySelectorAll()

Event Bubbling happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element. The primary use of event bubbling is the registration of program's built-in default functions. Recent years have seen a decline in the adoption of event capture and bubbles in particular by developers. Event bubbling is not required, but it may be difficult for users to keep track of the activities that are being carried out as a result of an event.

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action. A callback's primary purpose is to execute code in response to an event. These events might be user-initiated, such as mouse clicks or typing. With a callback, we may instruct our application to execute this code every time the user clicks a key on the keyboard.
